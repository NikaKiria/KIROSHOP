# KIROSHOP TECHNICAL DESCRIPTION

### Short Description:

- KIROSHOP is a simple ecommerce admin panel api built using express.

### Used Technologies:

- express
- mysql2
- dotenv
- jwt
- joi
- escape-html
- bcrypt
- eslint
- prettier

### UX

- User
  - Register
  - Login
  - Edit User Info
  - Delete user
- Order
  - Create Order
  - Get Orders
  - Delete Order
- Product
  - Create Product
  - Edit Product
  - Delete Product
  - Get Products
